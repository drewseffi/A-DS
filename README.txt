To use this software you must run on the Visual Studio Developer Command Prompt
To compile the game use the command "cd" to move to your folder destination
For example ("cd C:\Users\drews\game")
Then you use the "cl" command to compile the program which builds the .obj and .exe
For example ("cl cw.c")
After that just type "cw" to run the program. Now you have entered the game and can
follow the input prompts given to you by the program, there will be a short introduction
text that will explain how to operate the game
